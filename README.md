
This repository contains a collection of Python scripts demonstrating foundational computer vision techniques for edge detection using OpenCV, including implementations of the Sobel, Prewitt, and Canny algorithms optimized for applications like medical imaging and road lane detection. To run these tools locally, users must install Python along with the opencv-python, numpy, and matplotlib libraries. Because these scripts are designed for standard local execution rather than Google Colab environments, users simply need to clone the repository, update the image_path variable in the desired script to point to their own local image file, and execute the code to generate a side-by-side Matplotlib visual comparison of the original grayscale image and the resulting edge detection.
